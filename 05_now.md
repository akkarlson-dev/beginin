## IP Paragraph: Timestamp Record
*Drafted May 2026.*

This describes the architecture behind iN, a personal AI platform developed by Amy K. Karlson.

**Core claim:** iN builds a personal model from verified intentional signals: deliberate saves, notes, tags, prompt invocations, and behavioral confirmations: rather than passive behavioral data aggregated across users. Each act carries declared metadata; return visits confirm relevance. The accumulated signal produces a high-fidelity model of a specific user's values, interests, and evolving identity.

**The query model:** any experience built on iN takes two inputs: a *data bundle* (the personal model) and an *intention* (the experience requested in the moment). The bundle provides personalization; the intention specifies format. This enables downstream products to produce genuinely personal outputs without user configuration at query time.

**The prompt marketplace:** a discoverable library of shareable methodologies applied to private user data. Data stays private; only the prompt travels. Community layer enables discussion of outputs without exposing source material.

**The agent thesis:** a sufficiently rich personal model constitutes a definition of self precise enough to act as the substrate for a personal AI agent: one that embodies values rather than executes instructions.
